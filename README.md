# Waybackunifier-js

Get urls for the domain using wayback api implemented in nodejs.

To Run

## To Run
Clone this repository

```
$ npm install
$ node waybackunifier google.com true
```

    *google.com* - the domain which we want get urls.

    *true* - to include subdomain.
